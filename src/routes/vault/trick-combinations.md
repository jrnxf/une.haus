# Unicycle Trick Combinations

## Base Tricks

### Unispins

| Rotation | Name         |
| -------- | ------------ |
| 180°     | 180 unispin  |
| 360°     | 360 unispin  |
| 540°     | 540 unispin  |
| 720°     | 720 unispin  |
| 900°     | 900 unispin  |
| 1080°    | 1080 unispin |
| 1260°    | 1260 unispin |

### Crankflips

| Rotations | Name       |
| --------- | ---------- |
| 1         | crankflip  |
| 2         | doubleflip |
| 3         | tripleflip |
| 4         | quadflip   |
| 5         | quintflip  |
| 6         | sexflip    |

## Modifiers

- **Side** - Add "side" to make a sidespin variant (e.g., 180 sidespin, 360 sidespin)
- **Back** - Use a reverse crankflip (backflip) instead of regular crankflip

## Unispin + Flip Combinations

When combining a unispin with crankflips, special names are used:

| Unispin | + Flip    | + Doubleflip    | + Tripleflip    | + Quadflip    |
| ------- | --------- | --------------- | --------------- | ------------- |
| 180°    | hickflip  | hickdoubleflip  | hicktripleflip  | hickquadflip  |
| 360°    | treyflip  | treydoubleflip  | treytripleflip  | treyquadflip  |
| 540°    | fifthflip | fifthdoubleflip | fifthtripleflip | fifthquadflip |
| 720°    | sejflip   | sejdoubleflip   | sejtripleflip   | sejquadflip   |
| 900°    | ninthflip |                 |                 |               |
| 1080°   | tenthflip |                 |                 |               |

## Adding Modifiers

Modifiers can be combined with any trick:

### Side Variants

- hicksideflip
- treysideflip
- fifthsideflip
- sejsideflip

### Back Variants (reverse crankflip)

- hickbackflip
- treybackflip, treydoublebackflip, treytriplebackflip
- fifthbackflip
- sejbackflip

### Combined Modifiers

You can stack both side and back modifiers:

- hicksidebackflip
- treysidebackflip
- treysidedoublebackflip

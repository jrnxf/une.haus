## overview

une.haus has site-wide real-time chat. all riders share one conversation — no channels, no threads, just one place to talk.

## features

- **rich text** — format messages with bold, italic, links, and more
- **[@mentions](/docs/community/mentions)** — tag other riders to get their attention
- **likes** — react to messages with a like
- **[flagging](/docs/community/reporting)** — flag messages that break community guidelines

## catching up

scroll back through history with infinite scroll. if someone links you to a specific message, chat will jump directly to it and highlight it. a "jump to latest" button appears when you're viewing older messages so you can get back to the bottom quickly.

## real-time

messages appear instantly. no page refreshes needed.

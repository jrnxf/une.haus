## elements

elements are the core building blocks of tricks -- things like unispin, crankflip, and rolling wrap. you can browse the full list on the glossary page.

## modifiers

modifiers are qualifiers applied to elements. things like rolling, blind, late, or zero. they change how an element is performed.

## how they connect

tricks are composed of elements with modifiers applied. the glossary helps you understand what each part of a trick name means and how they combine.

## contributing

you can propose new elements and modifiers too. see [contributing tricks](/docs/tricks/contributing) for details.

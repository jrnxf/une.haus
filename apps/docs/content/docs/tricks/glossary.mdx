## elements

elements are the core building blocks of tricks — things like unispin, crankflip, and rolling wrap. adding or removing an element changes the trick's identity. the glossary page lists every element with its description.

## modifiers

modifiers describe how an element is performed without changing the trick itself. things like rolling, blind, late, or zero. a "crankflip" and a "rolling crankflip" use the same element but differ by modifier.

## browsing

the glossary page has two tabs — one for elements and one for modifiers. each entry shows its name and description.

## how they connect

tricks are composed of elements with modifiers applied. the glossary helps you understand what each part of a trick name means and how they combine.

## contributing

you can propose new elements and modifiers from the glossary page. see [contributing tricks](/docs/tricks/contributing) for details on the review process.

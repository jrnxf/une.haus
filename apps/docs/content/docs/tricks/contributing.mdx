title: contributing tricks
description: how to submit new tricks, propose updates, and add videos
---

## submitting a new trick

navigate to the create page from the tricks section. fill in the trick details: name, description, elements, and modifiers.

## proposing updates

on any trick detail page, you can suggest changes. this creates a diff that shows exactly what you're proposing, which goes to admin review.

## adding videos

submit a video to associate it with a trick entry. upload a clip or paste a link so other riders can see the trick in action. videos show up on the trick detail page alongside the description and metadata.

## proposing glossary additions

you can suggest new [elements or modifiers](/docs/tricks/glossary) to expand the glossary.

## the review process

all contributions go through admin review. quality stays high while the doors stay open.

see also: [contributing](/docs/contributing) for the broader philosophy on how riders shape the platform.

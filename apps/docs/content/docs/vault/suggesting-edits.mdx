## what you can suggest

title corrections, discipline tags, and rider attribution. if a video is mislabeled or missing a rider credit, this is how you fix it.

## how to suggest

from any video detail page, use the suggest action to propose changes. your suggestion creates a diff that goes to admin review.

## review process

all suggestions go through admin review before they're applied. this keeps the archive accurate while making it easy for anyone to contribute.

see also: [contributing](/docs/contributing) for the broader philosophy on community contributions.

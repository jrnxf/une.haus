## overview

une.haus has full tournament support with prelims, rankings, and bracket stages. each tournament has a public page that updates in real time.

## creating a tournament

head to the create page to set up a new tournament. a 4-digit join code is generated automatically so riders can join.

## joining

enter the 4-digit code to join a tournament.

## your tournaments

your tournament list shows all the competitions you're part of.

## running a tournament

if you're organizing, see [running a tournament](/docs/tournaments/running) for the full breakdown of phases and admin controls.

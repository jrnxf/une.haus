## phases

tournaments move through four phases:

- **prelims** -- the initial round where riders compete to qualify
- **ranking** -- riders are ordered based on prelim results
- **bracket** -- single-elimination matches until a winner is crowned
- **complete** -- the tournament is over and the winner is declared

## admin controls

the tournament creator manages phase transitions. this includes managing prelims, setting rankings, and advancing bracket matches.

## live view

every tournament has a public page with real-time updates, a split timer for matches, and a visual bracket display.

## bracket

the bracket visualization shows all matches and results as the tournament progresses.

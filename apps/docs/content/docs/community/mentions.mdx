## how to mention

type `@` and start typing a name. a dropdown appears with matching users. select one to insert the mention.

## where mentions work

chat, posts, comments, bios -- any text area on the site supports mentions.

## what happens

mentions render as clickable links to the mentioned rider's profile. the rider also gets a [notification](/docs/community/notifications).

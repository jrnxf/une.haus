## user profiles

each rider has a profile with avatar, bio, disciplines, social links, and an activity timeline showing their posts, game sets, submissions, and more.

## user directory

browse all riders on the platform. filter by name and disciplines with infinite scroll.

## user map

the map shows rider locations around the world. set your [location in your profile](/docs/getting-started/profile) to appear on it.

## online indicator

the sidebar shows how many riders are currently active. click the count to see a dropdown listing who's online.

## more

- [following](/docs/community/following) -- follow riders you care about
- [mentions](/docs/community/mentions) -- tag other riders in any text area
- [notifications](/docs/community/notifications) -- in-app notifications, email digests, game reminders
- [reporting content](/docs/community/reporting) -- flag content for moderation

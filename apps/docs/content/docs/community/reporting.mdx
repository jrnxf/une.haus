description: how to flag content for moderation
---

## what you can report

you can flag posts, game sets, game submissions, and messages across the platform. if something breaks the community guidelines, flagging it sends it to admin review.

## how to flag

on any flaggable piece of content, use the flag action (available in the menu or as a button). a dialog opens where you can describe the reason for the report. submit it and an admin will review it.

## what happens next

admins see all unresolved flags in the admin review panel. they can either remove the flagged content or dismiss the flag. you'll receive a notification when your flag is resolved with the outcome.

## limits

you can only flag a given piece of content once. duplicate flags on the same item are prevented.

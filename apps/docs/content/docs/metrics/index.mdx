## overview

the metrics page shows how the community is growing and who's contributing the most.

## dashboard

the main view displays six headline stats:

- **users** — total registered riders
- **countries** — how many countries are represented (links to the [user map](/docs/community))
- **posts** — total posts shared
- **messages** — total messages across chat, posts, game sets, and submissions
- **likes** — total likes across all content types
- **videos** — total video uploads

below the stats, two charts visualize activity:

- **activity by month** — a line chart tracking posts, messages, sets, and submissions over time
- **discipline distribution** — a pie chart showing the breakdown of rider disciplines

## top contributors

the dashboard highlights the top five contributors ranked by a point system:

| action  | points |
| ------- | ------ |
| content | 5      |
| message | 2      |
| like    | 1      |

content includes game sets, submissions, posts, trick contributions, and vault video suggestions. messages include chat, comments on posts, sets, and submissions. likes count across all content types.

## contributor leaderboard

the full leaderboard at `/metrics/users` ranks every contributor in a sortable table. columns include rank, name, content count, message count, like count, and total points. click any row to visit that rider's profile.

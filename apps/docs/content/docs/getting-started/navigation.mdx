---
title: navigation
description: how to get around une.haus on desktop and mobile
---

## sidebar

the collapsible sidebar gives you links to every section: games, users, posts, chat, tricks, vault, tourney, metrics, arcade, and shop. toggle it with `Cmd/Ctrl+B`.

## mobile drawer

on mobile, navigation lives in a bottom drawer with haptic feedback on supported devices.

## command palette

press `Cmd/Ctrl+K` to open the command palette. it gives you fuzzy search for users and vault videos, quick page navigation, and theme switching -- all from your keyboard.

## online indicator

the sidebar footer shows how many riders are currently active. click it to see a dropdown listing who's online.

## theme

switch between light, dark, and system themes from the command palette or user menu.

## keyboard shortcuts

| shortcut     | action               |
| ------------ | -------------------- |
| `Cmd/Ctrl+K` | open command palette |
| `Cmd/Ctrl+B` | toggle sidebar       |

## the repository

une.haus is open source. the full codebase lives at [github.com/jrnxf/une.haus](https://github.com/jrnxf/une.haus).

## tech stack

TanStack Start, React, Drizzle, PostgreSQL, Tailwind CSS, Mux.

## reporting bugs

found a bug? [open a GitHub issue](https://github.com/jrnxf/une.haus/issues) with details about what happened and what you expected.

## contributing code

fork the repo, create a branch, make your changes, run `bun preflight` to check everything passes, and open a pull request.
